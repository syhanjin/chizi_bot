# 群内指令-group_command

指令 参数 功能
禁言 @某人 [时间] 禁言某人指定时间，默认为60s，@全体成员则全员禁言
解禁 解除禁言 @某人 解禁某人，@全体成员则全员禁言
踢出 @某人 将某人从群内踢出

# 私聊&群内互通指令-integrated_command

指令 参数 功能
kw 修改群关键字 编辑群关键字 群关键字 由发送消息后询问 生成一个控制台链接，编辑群关键字拦截中的关键字
kwrefresh 刷新关键字缓存 rekw 关键字刷新 无 刷新群关键字缓存
