# handles.image

# class Draw

# async openfont(font=None, fontsize=16)

  • 说明

​ 打开字体文件

  • 参数

​ path: 路径

​ fontsize: 字体大小，默认16

若path == None则默认使用上次的字体文件

# async putText(...)

  • 说明：

​ 在图像的画布上打印一行文字

  • 参数：

​ text: str | tuple: 文字，若传入元组则类似print打印方案

​ direction: int = 1: 坐标移动方向 -1:不移动，0:向下，1:向右

​ interval: int = 0: 间隔，往direction方向的偏移量

​ font: str = None: 字体文件路径, 默认采用上次字体

​ fontsize: int = 16: 字体大小

​ fill: tuple[int, int, int] | str = (0, 0, 0): 字体填充颜色

​ border: int = 0: 边框大小

​ borderFill: tuple[int, int, int] | str = (0, 0, 0): 边框填充颜色

# property pos

  • 返回当前画布坐标，[x, y]

可单独设置 obj.x, obj.y

# class ImgDraw

基类handles.image.Draw

打开图片并生成一张与图片等大的画布

请在对象创建完成后调用

await obj.init()

绘制时请调用 obj.draw

# async init()

初始化对象，包括图片打开和创建画布

# resize(...)

重置画布与图片大小，参数与PIL.Image.resize一致

# composite()

合成画布与原图片

# async save(path)

调用composite并将图片保存至指定路径
