# class User

需要事先接入MongoDB

import pymongo
client = pymongo.MongoClient('127.0.0.1', 27017)
db = client['qbot'] # 群数据库
userdb = client['user'] # 与网站对接

如需关闭网站对接 删除如下代码即可

userdb = client['user']
...
user = userdb.userdata.find_one({'qq': user_id})
self.user = None
if user is not None:
    self.user = user['user']

初始化需传入2个参数group_id与user_id

user = User(group_id, user_id)

完成后将会从数据库中获取信息好像__init__并没有使用async

作为一名QQ用户所拥有的数据并不会被获得，请在正常使用前调用传入数据

user.update_from_event(event)

# update_date_from_[event, msg, info]

成功返回True否则返回False

# add_favor

增加好感度，需要在在数据库中设置好好感等级数据

{
    'lvl': 0, # <int> 好感等级
    'max': 0, # <int> 该好感等级的最大好感值
    'label': '', # <str> 该好感等级的名称
    'attitude': '' # <str> 该好感等级下BOT对用户的态度
}
