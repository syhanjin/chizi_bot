# handles.message_builder

# welcome_card(...)

  • 说明

    生成一条欢迎卡片格式的 MessageSegment.json 消息

  • 参数

    text: str: 欢迎语

    icon: str = '': 图标的网址，若无则使用群头像

    tips: list[tuple[str, str] | list[str, str]] | tuple[str, str] | str = []: 卡片中的提示信息，若不为list则自动在外层包裹list

    • tuple | list: [text, title]
      • text: 提示文本, 标题默认为 温馨提示
      • title: 当添加该值且不为空，则标题将被覆盖
    • str: 提示文本，标题为 温馨提示

    buttons: list[tuple[str, str] | list[str, str]] | tuple[str, str] = []: 卡片中的按钮，若不为list则自动在外层包裹list

    • tuple: [name, action]
      • name: 按钮内容
      • action: 按钮所需跳转的网址

# image(path: str = None, b64: str = None)

  • 说明

    生成 MessageSegment.image 消息

    处理顺序 path > b64

  • 参数:

    path: 图片路径

    b64: 图片base64
